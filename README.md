# Mega IPTV (Auto-Built)

https://github.com/sarfaraj74-Tech/megaiptv-auto/actions/workflows/super-mega.yml/badge.svg?branch=main https://img.shields.io/github/last-commit/sarfaraj74-Tech/megaiptv-auto?logo=github

Auto-generated, cleaned, **deduplicated (by channel name)**, and **alphabetically sorted** IPTV playlists sourced from the public **iptv-org** project.  
• Country, Language, and Category lists are updated every 6 hours.  
• Offline channels are filtered using fast HTTP checks.  
• EPG: `https://iptv-org.github.io/epg/guides.xml`

> Browse the **HTML catalog**: **https://sarfaraj74-Tech.github.io/megaiptv-auto/**
